Version History

A complete record of every feature added, bug fixed, and engineering concept implemented since the project started.

v6.0Documentation & Educational PlatformJuly 2025MAJOR
New pages
  • Landing page — dedicated homepage with feature showcase, material/vehicle overview, and testimonials
  • Feedback page — on-site guestbook with star ratings, role tags, filter by type, and sort by date/rating
  • Changelog page — full version history showing sustained development
  • README.md — complete GitHub project documentation with equations, methodology, and quick-start
In-game additions
  • Guided lessons — 4 structured engineering lessons (beam failure, triangle strength, material selection, tension/compression)
  • Why did it fail? — AI-style failure analysis explaining which member failed, its structural role, and how to fix it
  • Quiz mode — 3 questions after each test asking about what just happened
  • Keyboard shortcuts modal — press ? for a full shortcut reference overlay
  • Night mode UI — sidebar panels now match canvas night mode when toggled
  • Mobile layout — responsive panels, touch-optimized buttons
  • Visit counter — tracks plays in localStorage
v5.0Physics & Content OverhaulJuly 2025MAJOR
  • 3 new vehicles — Bicycle (15kg), City Bus (12,000kg), Train (200,000kg)
  • Carbon Fiber material — $400/m, ultra-light, ultra-strong
  • Resonance system — Tacoma Narrows–style oscillation builds under traffic load
  • Foundation cracking — overloaded anchor points develop visual crack lines
  • Tension/compression display — blue tint for tension, orange for compression
  • 2 new levels — City Bridge (repair pre-built spans) and Quake Zone (auto-earthquakes)
  • Achievement system — 14 achievements with pop-up toast notifications
  • Replay mode — record and play back simulation frames
  • Share bridge URL — encode design into a shareable link
  • Per-material sound — wood groans, steel pings, carbon rings differently
  • Mobile touch / pinch zoom — two-finger pinch supported
v4.0Engineering Tools UpdateJuly 2025FEATURE
  • Grid snap — joints snap to 25px grid for precise truss construction
  • Bridge templates — load pre-built Pratt Truss, Warren Truss, Arch, Suspension, or Simple Beam
  • Force diagram overlay — compression/tension arrows on every beam during testing
  • Engineering Report — printable technical report with material breakdown and stress analysis
  • CSV stress export — download per-beam data for Excel/Python analysis
  • Design comparison — test two designs, compare side-by-side on 6 metrics
  • About page — full technical documentation with equations, material table, limitations
  • 12 challenges — 4 per level including No Screws, Speed Run, Wood Semi, The Impossible
v3.0Game FeaturesJuly 2025FEATURE
  • Undo system — up to 60 steps, Ctrl+Z shortcut
  • Symmetry mode — mirror beams across span center automatically
  • Copy left half — mirror existing left-side structure to right
  • Night mode — stars, moon, glow effects on stressed beams
  • Rain storm — +20% vehicle force, −15% break limit
  • Wind burst / Earthquake — lateral forcing events
  • Slow motion — 20% speed for detailed observation
  • Leaderboard — best scores stored per-level in localStorage
  • Save/load designs — name and save any design to localStorage
  • Live stats panel — vehicles crossed, max load, beams broken, peak stress
v2.0Simulation Core RewriteJuly 2025MAJOR
  • Replaced Matter.js with custom Verlet physics — stable for arbitrary truss topologies
  • Beam subdivision — each beam split into ~38px segments with free intermediate joints
  • Sag-based failure — failure measured by joint displacement from settled baseline
  • Settled baseline system — 120-frame settle window before break-checking starts
  • Vehicle gravity — vehicles fall into chasm when bridge fails
  • Beam inspector — click any beam during test for stress %, mode, cost
  • Cable material — tension-only constraint (compression skipped)
  • Tower anchors — suspension bridge anchor points above road level
  • Grand Canyon level — mid-span pier with extra anchor points
v1.0Initial ReleaseJune 2025INITIAL
  • Matter.js physics — rigid body simulation with constraint pins
  • 3 materials — Wood, Steel, Concrete
  • 4 vehicles — Sedan, Van, Box Truck, Semi
  • 2 levels — Narrow Gorge, River Crossing
  • Wave system — automatic vehicle waves (later replaced by manual spawning)
  • Stress color feedback — amber → red as beams approach failure
  • GitHub Pages deployment — live at notivanly.github.io/BridgeGame
